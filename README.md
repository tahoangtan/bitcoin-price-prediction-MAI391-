# bitcoin-price-prediction-MAI391-
Predict Bitcoin price using Linear Regression

## 📌 Overview
This project predicts Bitcoin prices using Linear Regression.

## 📊 Dataset
The dataset includes:
- Open, High, Low, Close prices
- Volume
- Technical indicators (MA7, MA21, RSI, Volatility)

## ⚙️ Method
- Linear Regression
- Loss function: Mean Squared Error (MSE)

## 📈 Results
The model captures general trends but struggles with high volatility.

## 🧠 Key Concepts
- Regression model
- Feature engineering
- Time series split

## 📂 Files
- `main.py`: source code
- `bitcoin.csv`: dataset
- `report.docx`: project report
